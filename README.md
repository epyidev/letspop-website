# Let's PoP!

Official website for Let's PoP! - A digital project development company.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- Vue 3 + TypeScript
- Tailwind CSS
- Vite
